How to use

Adjust the links if you move files into one folder.

If you place all pages in the same website folder, you should update the links so they match your final folder structure.

  • If all HTML files sit together, use links like about.html and contact.html.
  • If they stay in separate folders, keep the relative paths as written.
  • Replace placeholder contact details inside the Contact page before publishing.
